Background and Borders

The CSS background properties are used to define the background effects for elements.

The background-color property specifies the background color of an element.

  • red
  • yellow
  • green
Other background properties that can be used include:
  • background-image
  • background-repeat
  • background-attachment
  • background-position

The CSS border properties allow you to specify the style, width, and color of an element's border. The border-style property can have from one to four values (for the top border, right border, bottom border, and the left border). This allows you to do mixed borders.

Border styles include:

  • solid
  • dotted
  • dashed
  • double
  • groove
  • ridge
  • inset
  • outset
  • none
  • hiden
  • mixed
The border-radius property is used to add rounded borders to an element:
  • normal
  • round
  • rounder
  • roundest

Fonts and Text Decoration

The CSS font properties define the font family, boldness, size, and the style of a text.

The font family of a text is set with the font-family property. The font-family property should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font, and so on. Start with the font you want, and end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available. Note: If the name of a font family is more than one word, it must be in quotation marks, like: "Times New Roman".
